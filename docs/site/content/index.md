# Nottario

Nottario is an open-source, self-hosted service that coordinates human
developers and their AI agents. It speaks the same language to both: a
human-friendly web UI for people, an MCP server for agents, on the
same project and the same data.

Three domains, all in one binary:

- **Tasks.** A backlog with dependencies and atomic claim, no calendar
  dates. Kanban and Gantt views for humans; "what's next" queries for
  agents.
- **Markdown context.** A shared store of skills, docs and notes with
  versioning and optimistic concurrency. Replaces loose `.md` files
  scattered across laptops.
- **Architecture.** A navigable diagram of the product, maintained in
  textual form so agents can edit it.
